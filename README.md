# Operating Systems Fundamentals — CS3070, NPS

Labs and exercises from CS3070 (Operating Systems) at NPS. Covers OS concepts including threading, synchronization, memory management, and inter-process communication.

## Contents

| Directory | Description |
|-----------|-------------|
| `Milestone1/` | Threading fundamentals — trivial vs heavy workload comparison |
| `ATM_Problem/` | ATM synchronization problem (initial implementation) |
| `Milestone3/` | Milestone 3 documentation |
| `Milestone4/` | Semaphore refinement |
| `Milestone5/` | Final ATM simulation with delta-based update protocol |
| `E0`–`E6` | Exercise sets covering OS concepts |
| `Exam1_Prep/` | Exam preparation materials |

## ATM Simulation

The primary project simulates concurrent ATM transactions. See [os-concurrency-atm-simulation](https://github.com/jdev-02/os-concurrency-atm-simulation) for the organized code-only version with detailed documentation.
